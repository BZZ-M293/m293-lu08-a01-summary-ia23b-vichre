Learningunit 1: HTML Basics und Werkzeuge

In dieser Einheit lernen Sie die Grundlagen von HTML und die wichtigsten Werkzeuge kennen, die für die Webentwicklung benötigt werden.

<!-- Beispielcode für HTML Basics -->
<!DOCTYPE html>
<html>
<head>
<title>Meine erste Webseite</title>
</head>
<body>
<h1>Hello World</h1>
</body>
</html>

Learningunit 2: Aufbau HTML Datei

Diese Einheit behandelt die grundlegende Struktur einer HTML-Datei, einschließlich der Bedeutung von <!DOCTYPE>, <html>, <head>, und <body>.

<!-- Beispielcode für den Aufbau einer HTML-Datei -->
<!DOCTYPE html>
<html lang="de">
<head>
<meta charset="UTF-8">
<title>Beispielseite</title>
</head>
<body>
<p>Dies ist der Inhalt der Seite.</p>
</body>
</html>

Learningunit 3: Blöcke und Inhalte

Hier lernen Sie die verschiedenen Block- und Inline-Elemente in HTML kennen und wie sie verwendet werden.

<!-- Beispielcode für Blöcke und Inhalte -->
<div>
<h2>Ein Blockelement</h2>
<p>Dies ist ein Absatz innerhalb eines Div-Blocks.</p>
<span>Dies ist ein Inline-Element.</span>
</div>

Learningunit 4a: Coding Style

In dieser Einheit wird der allgemeine Coding Style in HTML behandelt, einschließlich der Einrückung und der Benennungskonventionen.

<!-- Beispielcode für Coding Style -->
<!DOCTYPE html>
<html lang="de">
<head>
<meta charset="UTF-8">
<title>Guter Coding Style</title>
</head>
<body>
<h1>Sauberer Code</h1>
<p>Achten Sie auf konsistente Einrückung.</p>
</body>
</html>

Learningunit 4b: Statischer Coding Style

Diese Einheit fokussiert sich auf den statischen Coding Style, der speziell für statische Webseiten gilt.

<!-- Beispielcode für statischen Coding Style -->
<!DOCTYPE html>
<html lang="de">
<head>
<meta charset="UTF-8">
<title>Statischer Coding Style</title>
</head>
<body>
<h1>Statischer Content</h1>
<p>Dieser Inhalt ist statisch und ändert sich nicht.</p>
</body>
</html>

Learningunit 5: HTTP und Adressen

In dieser Einheit lernen Sie die Grundlagen von HTTP und URLs, einschließlich der Verwendung der Methoden GET und POST sowie dem Verlinken von Adressen und Bildern.

<!-- Beispielcode für HTTP und Adressen -->
<!-- GET-Methode -->
<form action="/example" method="get">
<label for="query">Suche:</label>
<input type="text" id="query" name="query">
<button type="submit">Senden</button>
</form>
 
<!-- POST-Methode -->
<form action="/submit" method="post">
<label for="name">Name:</label>
<input type="text" id="name" name="name">
<button type="submit">Senden</button>
</form>
 
<!-- Bilder verlinken -->
<img src="https://example.com/bild.jpg" alt="Beispielbild">
<a href="https://example.com">Link zur Beispielseite</a>

Learningunit 6: Formularelemente

Diese Einheit behandelt die grundlegenden Formularelemente in HTML, wie Textfelder, Schaltflächen und Auswahlfelder.

<!-- Beispielcode für Formularelemente -->
<form>
<label for="name">Name:</label>
<input type="text" id="name" name="name">
<br>
<label for="email">E-Mail:</label>
<input type="email" id="email" name="email">
<br>
<label for="gender">Geschlecht:</label>
<select id="gender" name="gender">
<option value="male">Männlich</option>
<option value="female">Weiblich</option>
</select>
<br>
<button type="submit">Absenden</button>
</form>

Learningunit 7: Erweiterte Formularelemente

Hier lernen Sie erweiterte Formularelemente kennen, wie z.B. Dateiauswahl, Farbauswahl und Bereiche.

<!-- Beispielcode für erweiterte Formularelemente -->
<form>
<label for="date">Datum:</label>
<input type="date" id="date" name="date">
<br>
<label for="color">Farbe:</label>
<input type="color" id="color" name="color">
<br>
<label for="range">Bereich:</label>
<input type="range" id="range" name="range" min="0" max="100">
<br>
<label for="file">Datei hochladen:</label>
<input type="file" id="file" name="file">
<br>
<button type="submit">Absenden</button>
</form>

Learningunit 8: Navigation (sehr basics)

Diese Einheit behandelt die Grundlagen der Navigation auf einer Webseite, einschließlich der Verwendung von Links und Menüs.

<!-- Beispielcode für Navigation -->
<nav>
<ul>
<li><a href="#home">Home</a></li>
<li><a href="#about">Über uns</a></li>
<li><a href="#contact">Kontakt</a></li>
</ul>
</nav>
